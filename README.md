# phenomenal-realism-kr

20세기 초 현상즉실재론의 한국 유입 연구: 디지털 문헌학적 접근

## 연구 개요

이노우에 데쓰지로(井上哲次郎)의 『철학과 종교』(1915)와 이돈화의 『인내천요의』(1924) 간의 텍스트 관계를 디지털 문헌학 방법론으로 분석하는 프로젝트.

## 프로젝트 구조

```
phenomenal-realism-kr/
├── data/
│   ├── raw/              # 원본 텍스트 (OCR 결과물)
│   ├── processed/        # 전처리된 DB (엑셀, CSV)
│   └── analysis/         # 분석 결과물 (유사도 매트릭스 등)
├── docs/
│   ├── methodology.md    # 방법론 상세 기술
│   ├── preprocessing.md  # 전처리 과정 문서화
│   ├── TODO.md           # 작업 목록
│   └── changelog.md      # 수정 이력
├── notebooks/            # 분석 과정 (Jupyter 등)
├── src/                  # 재사용 가능한 코드
├── app/                  # Streamlit 발표용 앱
├── images/               # 시각화 결과물
└── references/           # 참고문헌, 메모
```

## 주요 분석 내용

1. **지식의 차용**: 과학적 권위의 투명한 수용
2. **논리의 내재화**: '철학'의 소거와 주체의 교체
3. **가치의 전복**: 제국주의 논리의 해체와 재구성

## 기술 스택

- Python 3.x
- Streamlit (발표용 웹앱)
- Pandas (데이터 처리)
- 한자어 토큰화 및 유사도 분석
