## I. 발견된 학술적 한계 (6가지 비평 중 1개 철회 → 5가지 유효)

### 1. 해석의 과잉 추론 (Interpretive Overreach)
**문제**: 텍스트 간 유사성/차이를 "의도적", "전략적", "슬그머니" 등의 표현으로 저자의 의도로 단정
- 텍스트 유사성은 다양한 원인 가능: 공통 출처, 시대적 담론, 번역 관행 등
- 저자의 의도는 텍스트 분석만으로 증명 불가

### 2. 방법론적 한계 미고지
**문제**: 분석 방법의 한계를 명시하지 않음
- 자카드 유사도의 특성과 한계
- 한자어 토큰화의 문제점 (동음이의어, 문맥 무시)
- 임베딩 모델의 시대적 편향

### 3. 통계적 근거 부재
**문제**: "높은 유사도"의 기준이 자의적
- 0.6306이 왜 "높은" 것인지 설명 없음
- 전체 분포 대비 위치 제시 안 함

### 4. 비교 기준선 부재
**문제**: 두 텍스트만 비교하여 특수성 입증 불가
- 동시대 다른 텍스트와의 유사도 미측정
- 우연적 유사성과 실질적 영향 구분 불가

### 5. 맥락 분류의 주관성
**문제**: '進化' 용법 차이 주장에 정량적 근거 부족
- "생물학적 vs 도덕적" 분류 기준 불명확
- 실제 용례 분포 데이터 미제시

**원래 문제 제기**: 이돈화의 직접 발언/서문 인용 없음

**선행 연구 (1차 자료에 해당)**:
- 허수(2025), 「'공간의 시간화'에 맞선 '재귀적 시간화'」, 『한국사론』 71
- 허수(2015), 「『개벽』의 종교적 사회운동론과 일본의 '종교철학'」, 『인문논총』 72(1)
- 허수(2011), 『이돈화 연구』, 역사비평사

**본 연구의 실제 위상**:
- 선행 연구: 영향 관계가 **있다**는 사실을 규명 (도식, 구분법 등 쉬운 지표 중심)
- 본 연구: 영향 관계의 **구체적 양상과 특성**을 규명 (전체 텍스트 정밀 비교)

**본 연구의 새로운 기여**:
1. "선택적 전유(Appropriation)"의 구체적 패턴 발견 (유사도 스파이크/절벽 현상)
2. 무엇을 취하고 무엇을 버렸는가의 선별 논리 추적
3. 동일 용어(進化, 良心 등)의 의미 변용 과정 정량적 분석

**결론**: "1차 자료 부족" 비평은 연구 목적을 오해한 것으로 철회함.
본 연구는 단순한 "정량적 보완"이 아니라, 선행 연구가 본격적으로 다루지 못한
**영향 관계의 실체와 특성**을 디지털 문헌학으로 새롭게 규명하는 작업임.
