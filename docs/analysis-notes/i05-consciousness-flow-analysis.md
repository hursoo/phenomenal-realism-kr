## 핵심 발견: 헤켈 의식론의 변용

### 이노우에 (C06-P12~P18): 헤켈의 6종 의식론

1. 人間本位意識論 (Anthropistische Theorie)
2. 神經本位意識論 (Neurologische Theorie)
3. 動物本位意識論 (Animalische Theorie)
4. 生物本位意識論 (Biologische Theorie)
5. **細胞本位意識論 (Cellulare Theorie)** ← 이돈화가 **생략**
6. 原子本位意識論 (Atomistische Theorie)

### 이돈화 (P02~P07): 5종 의식론으로 축약

1. 사람 本位 意識論
2. 神經 本位 意識論
3. 動物 本位 意識論
4. 生物 本位 意識論
5. 原子 本位 意識論

> 이돈화는 '細胞本位意識論'을 **의도적으로 생략**하고 5종으로 재구성함.
