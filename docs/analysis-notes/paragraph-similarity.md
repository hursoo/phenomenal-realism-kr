# 문단 단위 유사도 분석 (2026-01-23 재산출)

## 배경: 기존 수치의 문제

발표문에 인용된 유사도 수치(0.6306, 0.349, 0.5259)의 출처를 검증한 결과, **기존 crosstab 데이터와도 불일치**함을 발견함. 이에 문단 단위로 유사도를 처음부터 재산출하고, 검증 가능한 새로운 지표로 교체함.

| 항목 | 발표문 (출처불명) | 기존 crosstab | 비고 |
|------|------------------|---------------|------|
| C06×C03-S04 | 0.6306 | 0.4853 | 불일치 |
| C07×C03-S04 | 0.349 | 0.6348 | **방향도 반대** |
| C08×C03-S04 | 0.5259 | 0.3487 | 불일치 |

## 새 분석 방법론

### 1. 분석 단위
- **문단(Paragraph)** 단위로 비교
- 1915 텍스트: 626개 문단 (line_class='TEXT'만, STRUCT 제외)
- 1924 텍스트: 365개 문단 (line_class='TEXT'만, STRUCT 제외)
- 총 비교 쌍: **228,490개** (626 × 365)

### 2. 유사도 계산
- **한자어 자카드 유사도 (Jaccard Similarity)**
- 토큰화: 정규표현식 `r'[一-龥]{2,}'` (2자 이상 한자어)
- 공식: `|A ∩ B| / |A ∪ B|`

### 3. 임계값 설정
- **유의미한 참조 관계**: 유사도 ≥ 0.1
- 원시 데이터: 228,490개 쌍 중 0.1 이상은 135개
- 노이즈 제외 후: **111개 (상위 0.05%)** - 서수만 공유(7개) + 범용어만 공유(17개) 제외
- 이 임계값은 분포 분석을 통해 결정됨 (상위 0.1% 부근에서 급격한 밀도 감소)

### 4. 집계 방식 (하이브리드)
장-절 단위 매트릭스 생성 시 두 가지 지표 병용:
- **max**: 해당 장-절 조합 내 최대 유사도 (스파이크 탐지용)
- **count**: 해당 장-절 조합 내 유사도 ≥ 0.1인 쌍의 개수 (참조 밀도/커버리지)

| 구간 | 의미 | count (≥0.1) | max |

※ 노이즈 24개 제외 후 (서수 7개 + 범용어 17개)

### 3. '참조의 절벽' 현상
- C06(생명 정의): 23개 문단 쌍이 집중적 참조
- C07(나가이 비평): 0개로 **완전한 단절**
- C08(메치니코프): 1개 참조
- 해석: 이돈화가 '보편적 과학 지식'(C06)은 수용하되, '일본 학계 내부 논쟁'(C07)은 필터링

### 4. C06 × C03-S04의 23개 쌍 내용 분석
주로 두 영역에서 집중:
1. **의식론 분류** (C06-P12~P18 × C03-S04-I05): 헤켈의 6종 의식론 → 이돈화의 5종 의식론
2. **생명의 특징** (C06-P05, P08 × C03-S04-I04): 統一, 目的 등 생명 속성 정의
