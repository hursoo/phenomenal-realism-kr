## 2. 컬럼 구조

두 DB는 동일한 10개 컬럼을 공유함.

| 컬럼 | 타입 | 설명 | 예시 |
|------|------|------|------|
| `uid` | string | 고유 식별자 | BK_IT_1915_PR_00001 |
| `book_id` | string | 책 식별자 | BK_IT_1915_PR |
| `local_id` | string | 계층적 위치 ID | C01-S01-P01-S01 |
| `page_info` | string | 원본 페이지 번호 | 278 |
| `line_class` | string | 행 유형 | TEXT, STRUCT, ANNOTATION |

| `structure_id` | string | 소속 구조 ID | C01-S01 |
| `parent_id` | string | 상위 구조 ID | C01 |
| `depth_level` | int | 계층 깊이 | 0, 1, 2, 3, 10, 99 |
| `sort_order` | int | 정렬 순서 | 1, 2, 3, ... |

## 6. ID 체계 (local_id)

### 1915 철학과 종교

```
C##-P##-S##          # 절(Section)이 없는 장
C##-S##-P##-S##      # 절이 있는 장 (C01, C13, C14, C20)
```

- C##: Chapter (장)
- S##: Section (절) - 일부 장에만 존재
- P##: Paragraph (문단)
- S##: Sentence (문장)

### 1924 인내천요의

```
C##-S##-I##-P##-S##
```

- C##: Chapter (장)
- S##: Section (절)
- I##: Item (항)
- P##: Paragraph (문단)
- S##: Sentence (문장)
