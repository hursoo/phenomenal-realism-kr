# data/processed/

전처리가 완료된 구조화된 DB 파일을 보관하는 폴더입니다.

## 포함 대상

- 문장/문단 단위 DB (.xlsx, .csv)
- 한자어 토큰화 결과
- 메타데이터가 결합된 코퍼스

## 예상 파일

- `BK_IT_1915_PR_v1.1.xlsx` - 철학과 종교 문장 DB
- `BK_YD_1924_IY_v1.1.xlsx` - 인내천요의 문장 DB
- (문단 단위 DB 추가 예정)

## 파일 명명 규칙

```
BK_[저자코드]_[연도]_[텍스트코드]_v[버전].xlsx

- BK: Book
- IT: Inoue Tetsujiro (이노우에 데쓰지로)
- YD: Yi Don-hwa (이돈화)
- PR: Philosophy and Religion (철학과 종교)
- IY: Innaecheon Yoeuui (인내천요의)
```
