## 발견된 문제점


### 1. ⚠️ '철학' 빈도수 심각한 불일치 (가장 중요)

┌──────────────────┬─────────────────┬─────────────────┐
│ 위치             │ 이노우에(1915)  │ 이돈화(1924)    │
├──────────────────┼─────────────────┼─────────────────┤
│ II장 본문        │ 366회          │ 6회             │
│ IV장 본문        │ 582회          │ 15회            │
│ 인터랙티브 차트  │ 582회          │ 15회            │
└──────────────────┴─────────────────┴─────────────────┘

[문제]
같은 발표에서 이노우에 텍스트의 '철학' 출현 횟수가 366회와 582회로
서로 다르게 기술됨.

[권장 조치]
- 정확한 수치를 확인하여 통일 필요
- II장의 366회가 오류라면 수정
- 측정 기준이 다르다면 그 차이를 명시해야 함

[해당 위치]
- Line 462 (II장): "이노우에의 1915년 텍스트에서 '철학'은 366회나 등장"
- Line 585 (IV장): "원전인 이노우에의 텍스트에서 '철학(哲學)'은 총 582회나 호출"


--------------------------------------------------------------------------------

### 2. ⚠️ 동일 문단 중복 삽입 (복사 오류)

[위치]
II장 내 두 군데에 동일한 문장이 반복됨

- Line 364-366: "둘째, 반면 이돈화의 『인내천요의』는 총 6장으로 구성된
                 약 9만 4천 자..."
- Line 407-409: 동일 문장 그대로 반복

[권장 조치]
Line 407-409의 중복 문단 삭제


--------------------------------------------------------------------------------

### 3. ⚠️ 장(章) 번호 체계 혼란

┌─────────────────────┬──────────┬─────────────────────────┐
│ 본문 표현           │ ID 표기  │ 내용                    │
├─────────────────────┼──────────┼─────────────────────────┤
│ "제7장" (line 539)  │ -        │ 생명의 정의 (Bewegung)  │
│ "제6장" (line 570)  │ C07      │ 생명의 정의             │
│ "제7장" (line 571)  │ C08      │ 나가이 박사 학설 비평   │
└─────────────────────┴──────────┴─────────────────────────┘

[문제]
서장을 C01로 부여했다면 "제6장 = C07"이 맞는데,
line 539에서는 같은 내용을 "제7장"이라고 기술함.

[권장 조치]
장 번호를 일관되게 수정 (제6장/C07 또는 제7장/C08 중 하나로 통일)


--------------------------------------------------------------------------------

### 4. ⚠️ 유사도 수치 미세 불일치

- Line 538: "0.6306이라는 이례적인 유사도"
- Line 571: "0.635라는 이례적인 고유사도"

[문제]
같은 구간(생명 정의)의 유사도가 0.6306과 0.635로 다르게 표기됨.

[권장 조치]
소수점 셋째 자리까지 통일 (0.631 또는 0.635)


--------------------------------------------------------------------------------

### 5. △ '진화' 빈도 데이터 검증 필요

[본문 인용]
"이 단어는 이돈화의 텍스트에서 130회나 등장하며,
원전 대비 67%라는 이례적으로 높은 사용 빈도를 보인다."

[계산 검증]
130회가 67%라면 이노우에 텍스트에서는 약 194회(130÷0.67) 등장해야 함.

[권장 조치]
이노우에 텍스트의 '진화' 빈도(약 194회?)를 명시하거나, 시각화 자료로 보완


--------------------------------------------------------------------------------

### 6. △ '양심' 비교의 논증력 약화 우려

[본문 인용]
"본 연구가 수행한 문단 비교 분석(유사도 0.176)"

[문제]
0.176은 상당히 낮은 유사도로, 이는 비교 대상 문단 자체가 주제적으로
유사하지 않을 가능성도 있음. "가치의 분기"를 주장하려면, 먼저 두 문단이
같은 주제('양심')를 다루고 있음을 확인해야 논증이 성립함.

[권장 조치]
용례 버튼의 내용에서 두 문단이 실제로 '양심'을 정의하는 병렬적 구절임을 명시


--------------------------------------------------------------------------------

### 7. △ 이노우에의 '제국주의적' 진화론 해석 근거 미제시

[본문 인용]
"이노우에에게 진화는... 우수한 일본 민족의 지배를 정당화하는
약육강식의 법칙이었다."

[문제]
이 주장은 발표문 본문에 직접적인 텍스트 인용이 없음.
버튼(🔍진화론 재해석)을 눌러야 용례를 볼 수 있으나,
청중은 발표 중 이를 확인하기 어려움.

[권장 조치]
핵심 인용문 1-2문장을 본문에 직접 포함
