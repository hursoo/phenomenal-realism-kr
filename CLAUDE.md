# CLAUDE.md

## Project Overview

디지털 문헌학 연구 프로젝트: 이노우에 데츠지로의 『철학과 종교』(1915)와 이돈화의 『인내천요의』(1924) 간 텍스트 관계 분석.

서울대학교 국사학과 허수 교수의 개인 연구 프로젝트로, 2025년 2월 중순 정식 발표 예정.

## Commands

```bash
# Install dependencies
pip install -r app/requirements.txt

# Run the Streamlit presentation app
streamlit run app/app.py
```

## Architecture

### Key Components

- **app/app.py**: Streamlit 발표용 웹앱 (메인 결과물). 5개 섹션으로 구성:
  - I. 서론
  - II. 데이터 코퍼스 구축
  - III. 지식의 차용
  - IV. 논리의 내재화
  - V. 가치의 전복

- **data/**: 연구 데이터
  - `raw/`: OCR 결과물 원본
  - `processed/`: 전처리된 DB (엑셀)
  - `analysis/`: 유사도 분석 결과

- **app/data/**: 앱 전용 데이터 파일
  - `BK_IT_1915_PR_v1.3.xlsx`: 1915 텍스트 DB (11,088행)
  - `BK_YD_1924_IY_v1.2.xlsx`: 1924 텍스트 DB (2,254행)

- **docs/db_structure.md**: DB 구조 및 버전 이력 상세 문서

### Data Structure

코퍼스는 문장 단위로 구축됨.
- 1915: ID 체계 `C##-P##-S##` (장-문단-문장), C00(서장)~C28(부록2)
- 1924: ID 체계 `C##-S##-I##-P##-S##` (장-절-항-문단-문장), C01~C06

## Coding Guidelines

1. **역할**: 역사학적 맥락을 이해하는 연구 조교. 단순 코더가 아님.
2. **언어**: 모든 주석과 설명은 한국어로 작성. 정중하고 학술적인 어조.
3. **기술 스택**: Python 3.11, Streamlit, Pandas, Plotly.
4. **데이터 정확성**: 사료 원문 변형 최소화가 최우선.

## Research Context

분석의 3대 축:
1. **지식의 차용**: 과학적 권위의 투명한 수용
2. **논리의 내재화**: '철학' 개념의 소거와 주체 교체

### 허수(2025) 논문 반영 작업 완료

**참조 논문**: 허수(2025), 「'공간의 시간화'에 맞선 '재귀적 시간화' - 1920년 전후 개벽 주도층의 현상즉실재론 수용과 변용 -」, 『한국사론』 71

이 논문의 핵심 개념들을 발표 자료(app/app.py)에 반영함. 커밋: `6e79d7f`

#### 반영된 핵심 개념

1. **'공간의 시간화' (temporalization of space)**
   - 서구-일본을 '문명/진보'의 시점에, 식민지를 '야만/정체'의 시점에 배치
   - 공간적 지배를 시간적 우월성으로 정당화하는 제국주의적 메커니즘

2. **'재귀적 시간화' (recursive temporalization)**
   - 제국의 선형적 진보 시간관에 대한 대항 논리
   - '되돌아감(회귀)'을 통해 미래를 여는 역설적 시간성
   - 김기전의 '출발점-도착점' 담론과 연결

3. **'이데올로기적 모듈' (ideological module)**
   - 이노우에의 현상즉실재론이 맥락 이식 가능한 '모듈'로 기능
   - 형식적 틀(변증법 구조)은 유지하되 내용(제국주의 → 인내천)은 대체 가능

4. **도식 비교**
   - 『철학과 종교』 66쪽 vs 『인내천요의』 96쪽
   - '현상-실재'의 '상-하' 이원적 구조 전유
   - '철학적 절대자' → '한울님(天)'으로 대체

## 문단 단위 유사도 분석 (2026-01-23 재산출)

### 배경: 기존 수치의 문제

발표문에 인용된 유사도 수치(0.6306, 0.349, 0.5259)의 출처를 검증한 결과, **기존 crosstab 데이터와도 불일치**함을 발견함. 이에 문단 단위로 유사도를 처음부터 재산출하고, 검증 가능한 새로운 지표로 교체함.

| 항목 | 발표문 (출처불명) | 기존 crosstab | 비고 |
|------|------------------|---------------|------|
| C06×C03-S04 | 0.6306 | 0.4853 | 불일치 |
| C07×C03-S04 | 0.349 | 0.6348 | **방향도 반대** |
| C08×C03-S04 | 0.5259 | 0.3487 | 불일치 |

### 새 분석 방법론

#### 1. 분석 단위
- **문단(Paragraph)** 단위로 비교
- 1915 텍스트: 626개 문단 (line_class='TEXT'만, STRUCT 제외)
- 1924 텍스트: 365개 문단 (line_class='TEXT'만, STRUCT 제외)
- 총 비교 쌍: **228,490개** (626 × 365)

#### 2. 유사도 계산
- **한자어 자카드 유사도 (Jaccard Similarity)**
- 토큰화: 정규표현식 `r'[一-龥]{2,}'` (2자 이상 한자어)
- 공식: `|A ∩ B| / |A ∪ B|`

#### 3. 임계값 설정
- **유의미한 참조 관계**: 유사도 ≥ 0.1
- 근거: 228,490개 쌍 중 0.1 이상은 **135개 (상위 0.06%)**
- 이 임계값은 분포 분석을 통해 결정됨 (상위 0.1% 부근에서 급격한 밀도 감소)

#### 4. 집계 방식 (하이브리드)
장-절 단위 매트릭스 생성 시 두 가지 지표 병용:
- **max**: 해당 장-절 조합 내 최대 유사도 (스파이크 탐지용)
- **count**: 해당 장-절 조합 내 유사도 ≥ 0.1인 쌍의 개수 (참조 밀도/커버리지)

### 핵심 분석 결과

#### 3. '참조의 절벽' 현상
- C06(생명 정의): 24개 문단 쌍이 집중적 참조
- C07(나가이 비평): 0개로 **완전한 단절**
- 해석: 이돈화가 '보편적 과학 지식'(C06)은 수용하되, '일본 학계 내부 논쟁'(C07)은 필터링

#### 4. C06 × C03-S04의 24개 쌍 내용 분석

1. **의식론 분류** (C06-P12~P18 × C03-S04-I05): 헤켈의 6종 의식론 → 이돈화의 5종 의식론
2. **생명의 특징** (C06-P05, P08 × C03-S04-I04): 統一, 目的 등 생명 속성 정의
